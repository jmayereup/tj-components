Teacher Jake Components

This domain hosts a collection of interactive language learning web components. These scripts can be easily embedded into blogs, websites, or LMS platforms to create rich educational experiences.

Available Scripts

Include any of these scripts in your HTML using a <script type="module" src="..."> tag.

  • Speed Review (tj-speed-review) tj-speed-review.js
  • Grammar Hearts (tj-grammar-hearts) tj-grammar-hearts.js
  • Pronunciation (tj-pronunciation) tj-pronunciation.js
  • Info Gap (tj-info-gap) tj-info-gap.js
  • Listening (tj-listening) tj-listening.js
  • Reader (tj-reader) tj-reader.js
  • Chapter Book (tj-chapter-book) tj-chapter-book.js
  • Quiz Element (tj-quiz-element) tj-quiz-element.js

Example Usage

<script type="module" src="https://scripts.teacherjake.com/tj-reader.js"></script>
<tj-reader>...</tj-reader>
